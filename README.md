# CTFd Media Library

A CTFd plugin that adds a media library to the admin panel for browsing, uploading, and deleting files.

## Features

- Dedicated **Media** page in the admin menu
- Browse all CTFd files in a visual grid (thumbnails for images, icons for other types)
- Upload one or multiple files at once
- Search and filter by type (images, PDF, archives, other)
- Click a file to get its full or short URL with a one-click copy button
- Image lightbox preview
- Delete files directly from the interface
- Optional floating shortcut icon on all admin pages

## Installation

Copy the `ctfd-media-library-main` folder into your CTFd `plugins/` directory:

```
CTFd/plugins/ctfd-media-library-main/
```

Restart CTFd. A **Media** entry will appear in the admin plugin menu.

## Dependencies

- CTFd >= v3.x
- Compatible with Docker and local installations
- An up-to-date browser with JavaScript enabled
- CTFd theme: Core-beta
